## 2. Signature features per competitor (steal-worthy)

- **Bloomberg**: command line + function codes; ticker strip; modular homepage; Big Take; integrated TV; ECO calendar; MSG culture. → AKASH's core identity.
- **Reuters/Eikon**: wire-speed headlines, 3-point summaries, sourcing discipline. → KeyPoints block; strict attribution.
- **CNBC**: live TV hero placement, pre/post-market framing ("Futures point to..."). → session-aware ticker labels, TV module.
- **WSJ/FT**: typography, standfirsts, columnist branding, newsletter empire. → article design + doc 29.
- **TradingView**: lightweight-charts OSS, indicator UX, compare overlays, symbol page layout. → GP function.
- **Koyfin**: saved dashboard views, watchlist analytics, elegant dark data-viz. → saved layouts, W design.
- **Yahoo Finance**: SEO quote pages, portfolios that regular people actually use, per-ticker RSS. → /quote pages + PORT simplicity.
- **Investing.com**: economic calendar with importance stars + actual/forecast/previous columns. → ECO design.

## 3. AKASH differentiators

1. **Bloomberg-grade terminal, free, in the browser** — nobody offers command-line + panels free.
2. **News + terminal in one product** — peers do one or the other.
3. **Hindi-first finance** — underserved, huge market (doc 35).
4. **Radical data honesty** — visible data age, source labels, "unavailable" instead of stale/fake values. Turn the free-data constraint into a trust feature.
